0x01 简介


PHP verion 8.1.0-dev于2021年3月28日与后门一起发布，但是后门很快被发现并删除。


0x02 漏洞概述


PHP verion 8.1.0-dev的PHP在服务器上运行，则攻击者可以通过发送User-Agentt标头执行任意代码。


0x03 影响版本


PHP 8.1.0-dev
